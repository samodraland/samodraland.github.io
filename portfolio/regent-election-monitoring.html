Regent Election Monitoring System

Portfolio

As part of a project aimed at ensuring transparency and fairness in a local regent election, I developed two integrated applications: a responsive web platform and a hybrid mobile app. These tools were designed specifically to support election monitoring efforts, empowering one of the candidates and their team with real-time insights during the electoral process.

The primary purpose of the system was to track and verify voting data from various polling stations as results came in. By allowing the team to compare their own vote counts with those of their opponents, the platform played a critical role in identifying discrepancies early, enabling faster response and follow-up. The mobile app, built using hybrid technologies, was optimized for field use—ensuring that on-ground volunteers and observers could quickly report data and incidents from remote locations.

Beyond vote tracking, the application also served as a valuable decision-support tool. The data it provided helped the candidate’s strategists analyze regional voting trends and formulate tactical adjustments both during and after the election.

Language: Javascript, PHP
Frontend Framework: Vue 2, Quasar Framework
Backend Framework: Snappy
Mobile: Cordova
Database: MySQL
